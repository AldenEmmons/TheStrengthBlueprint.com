# The System

> The structural logic behind how our programs are built.
> All workout plans must follow this system — not a generic template.

## Program Logic

Every program follows the Foundation → Strengthen → Perform arc.
No one skips Phase 1. Motor control is always established before progressive overload.

## Phase Breakdown

### Phase 1 — Foundation
**Goal:** Reduce fear-avoidance, settle inflammation, build proximal stability.
- Pain Neuroscience Education (PNE): 100–400 minutes across weeks
- Isometrics for the affected area (reduces cortical inhibition)
- Core and proximal stability work
- No heavy loading yet

### Phase 2 — Strengthen
**Goal:** Progressive loading with clear tolerance rules.
- Movement progression: bilateral → unilateral
- Range of motion progression: partial → full
- Pain tolerance rule: up to 3–5/10 acceptable if returns to baseline within 24 hours
- Graded Exposure applied throughout

### Phase 3 — Perform
**Goal:** Sport-specific or goal-driven output with objective criteria.
- Programming tied to the client's actual goals (sport, lifestyle, aesthetic)
- Objective progression metrics — nothing is left to feel

## Progression Rules

1. Proximal stability must be established before distal loading
2. Motor control must be demonstrated before adding strength load
3. Pain above 5/10 or not returning to baseline within 24 hours = back off load
4. Bilateral before unilateral, partial ROM before full ROM

## What Makes a Plan "Ours"

- It addresses the fear-avoidance pattern directly
- It educates the client on why the program works (PNE)
- It has clear phase structure with objective criteria to advance
- It never tells the client to just stop and rest
- It is specific — exercise selection, sets, reps, progression rules are explicit
